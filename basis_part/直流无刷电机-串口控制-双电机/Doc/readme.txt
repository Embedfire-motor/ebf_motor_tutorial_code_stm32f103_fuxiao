【*】 程序简介 
-工程名称：直流无刷电机-串口控制-双电机
-实验平台: 野火STM32 F103 拂晓开发板
-MDK版本：5.30
-HAL库版本：V1.1.4

【 ！】功能简介：
通过串口发送指令,设置电机方向和速度.
指令示例:
d1 0
电机1方向正转
v1 1000
电机1速度设置为1000
注意指令结尾需要换行,或在串口助手中勾选发送新行,一次只能发送一条指令

【 ！】实验操作：

无刷电机驱动接口一接线:
电机驱动板 5V_IN\GND 		<----> 开发板 5V\GND
电机驱动板 U+\V+\W+ 		<----> 开发板 PA15\PB3\PB10
电机驱动板 U-\V-\W- 		<----> 开发板 PA7\PB0\PB1
电机驱动板 SD\GND				<----> 开发板 PC9\GND
电机驱动板 HU\HV\HW\GND	<----> 开发板 PA15\PB3\PB10\GND --> 编码器

无刷电机驱动接口二接线:
电机驱动板 5V_IN\GND 		<----> 开发板 5V\GND
电机驱动板 U+\V+\W+ 		<----> 开发板 PE9\PE11\PE13
电机驱动板 U-\V-\W- 		<----> 开发板 PE8\PE10\PE12
电机驱动板 SD\GND				<----> 开发板 PD15\GND
电机驱动板 HU\HV\HW\GND	<----> 开发板 PD12\PD13\PD14\GND --> 编码器

下载本程序，复位开发板即可。

【*】 引脚分配

定时器TIM1输出PWM信号:
三路PWM输出到PE9\PE11\PE13上连接到驱动板上的U+\V+\W+接口.

定时器TIM8输出PWM信号:
三路PWM输出到PA15\PB3\PB10上连接到驱动板上的U+\V+\W+接口.

电机使能引脚:
电机驱动板无刷电机驱动接口一 SD <----> 开发板 PC9
电机驱动板无刷电机驱动接口二 SD <----> 开发板 PD15

编码器:
霍尔编码器信号从驱动板HU\HV\HW引脚输出,接入开发板。
（电机驱动板无刷电机驱动接口一）PD12\PD13\PD14,由定时器4捕获.
（电机驱动板无刷电机驱动接口一）PA15\PB3\PB10,由定时器2捕获.
